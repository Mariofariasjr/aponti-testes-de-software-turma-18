## 📖 Sobre o Projeto

Bem-vindo ao repositório da **Formação em Testes de Software**!

Este projeto faz parte do curso **APONTI** e documenta minha jornada de aprendizado em testes de software, desde os conceitos básicos até a automação avançada.

> *"Qualidade não é um ato, é um hábito."* — Aristóteles

<!-- ==================== OBJETIVOS ==================== -->
## 🎯 Objetivos

- ✅ Construir um portfólio profissional
- ✅ Dominar Git e GitHub
- ✅ Aprender testes manuais e automatizados
- ✅ Desenvolver habilidades em API Testing
- ✅ Implementar integração contínua
